# Rommel Rapada - Portfolio

Live Web Experience: [rommel.pages.dev](https://rommel.pages.dev)

A modern, high-performance, cinematic personal portfolio designed to bridge traditional pedagogy with modern digital innovation.

## Features
- **GSAP & Lenis Integration**: Buttery-smooth custom scrolling and scroll-triggered reveals that ensure fluid narrative direction.
- **WebGL Hero Canvas**: Responsive dynamic gradient shader that breathes with interaction.
- **Magnetic UI Hooks**: Micro-interactions spanning from work cards to trailing cursors, elevating presentation mechanics.
- **Matter.js Sandbox**: A gravity-based physics sandbox embedded into the footer. Pull on the letters to see them fly.
- **100% Client-Side Setup**: No build steps required. Ever. Native standard technologies working at their best.

## Structure
- `index.html` - The single-page application containing all markup, styles, and interconnected logic paths.
- `assets/` - Contains all imagery, system favicons, and Webclips.

## Deployment
This project is configured natively as a static site and is ready to deploy directly via Cloudflare Pages, Vercel, Route 53, or GitHub Pages. No extra build command is needed.

## Configuration & Content Maintenance
All text content, section arrangements, and color systems are gracefully tokenized. You can freely edit `index.html`'s `:root` CSS block to tweak colors, or update the semantic JSON-LD schema for immediate SEO enhancements.
